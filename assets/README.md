# Vendored assets

## IBMPlexMono-Regular.ttf

IBM Plex Mono (Regular), used by `brainscan.font.generate_lane_font_atlas`
to render the antialiased text in the bottom conversation strip.

- Source: <https://github.com/google/fonts/tree/main/ofl/ibmplexmono>
- Version: 2.3
- Licence: SIL Open Font License 1.1 (see `OFL.txt`)

The font is vendored into the repository so the dev machine and the Jetson
render identical glyphs regardless of which fonts the OS has installed.
